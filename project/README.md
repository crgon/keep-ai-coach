### 基于LangChain的AI运动教练助手
#### 最少功能（MVP - 3月底必须完成）
1. 核心对话功能
  - 使用LangChain的LLMChain实现基础问答
  - 支持运动相关问题的回答（如减脂、增肌建议）
    
2. 多轮对话支持
  - 集成ConversationBufferMemory
  - 能够记住最近5轮对话历史
    
3. 工具调用能力
  - 实现1个工具：运动强度计算器
  - 输入：身高、体重、年龄、运动频率
  - 输出：建议运动强度（低/中/高）
    
4. 基础前端界面
  - Streamlit简单界面
  - 输入框 + 发送按钮 + 对话显示区域
    
5. 文档与部署
  - 完整的README.md（项目介绍、安装步骤、使用说明）
  - 代码上传到GitHub
  - 本地可运行Demo
